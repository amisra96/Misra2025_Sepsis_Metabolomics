
# Misra2025_Sepsis_Metabolomics

This repository contains the **metabolomics data analysis** accompanying our BioRxiv submission:

**Title:** "Branched-chain ketoacid antioxidants mediate disease tolerance to sepsis"  
**Link to manuscript:** [https://www.biorxiv.org/content/10.1101/2025.05.22.654748v1](https://www.biorxiv.org/content/10.1101/2025.05.22.654748v1)

Metabolomics were performed on **liver tissues from mice** that either **survived or did not survive** sepsis. The goal was to identify **metabolic signatures associated with survival** and understand the biochemical pathways underlying **disease tolerance**.

---

## ğŸ”¬ Analysis

- **Machine learning approaches** were applied to the metabolomics dataset to:
  - Identify **differential metabolites** between survivors and non-survivors
  - Build predictive **classifiers** to distinguish the two groups
- Multiple ML techniques were used, including (but not limited to) **random forests, support vector machines, and logistic regression**.
- **Pathway analysis** was performed to highlight metabolic pathways that were **enriched or depleted** in survivors versus non-survivors.
